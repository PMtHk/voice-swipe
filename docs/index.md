# Voice Swipe

음성 명령으로 인스타그램 릴스를 넘기고 볼륨을 조절하는 크롬 익스텐션.

요리 중이거나 손을 쓸 수 없는 상황에서 "다음", "음소거", "소리 키워" 한 마디로 컨트롤하세요.

---

## 기능

- 🎤 **음성 명령**: 네비게이션 + 볼륨/음소거 제어
- 🌏 **한국어 / 영어 지원**
- 🔒 **개인정보 수집 없음**: 모든 인식은 브라우저 내에서 처리
- 🔋 **배터리 최적화**: 탭 비활성 시 자동 중지
- ⚙️ **민감도 조절**: 환경에 맞게 인식 임계값 설정

## 지원 플랫폼

- Instagram Reels (`instagram.com/reels/*`)

## 지원 명령어

**네비게이션**
- "다음" / "next"
- "이전" / "back" / "뒤로"

**볼륨**
- "음소거" / "mute"
- "소거해제" / "unmute"
- "소리 키워" / "volume up"
- "소리 줄여" / "volume down"
- "소리 최대" / "max volume"
- "소리 최소" / "min volume"

## 사용법

1. 크롬 웹스토어에서 Voice Swipe 설치
2. 인스타 릴스 페이지 접속
3. 확장 아이콘 클릭 → 마이크 켜기
4. 원하는 명령어 발화
