# Voice Swipe

음성 명령으로 유튜브 쇼츠와 인스타그램 릴스를 넘기는 크롬 익스텐션.

요리 중이거나 손을 쓸 수 없는 상황에서 "다음" 한 마디로 영상을 넘겨보세요.

---

## 기능

- 🎤 **음성 명령**: "다음" / "next", "이전" / "back"
- 🌏 **한국어 / 영어 지원**
- 🔒 **개인정보 수집 없음**: 모든 인식은 브라우저 내에서 처리
- 🔋 **배터리 최적화**: 탭 비활성 시 자동 중지
- ⚙️ **민감도 조절**: 환경에 맞게 인식 임계값 설정

## 지원 플랫폼

- YouTube Shorts (`youtube.com/shorts/*`)
- Instagram Reels (`instagram.com/reels/*`)

## 사용법

1. 크롬 웹스토어에서 Voice Swipe 설치
2. 유튜브 쇼츠 또는 인스타 릴스 페이지 접속
3. 확장 아이콘 클릭 → 마이크 켜기
4. "다음" 또는 "이전"이라고 말하기
